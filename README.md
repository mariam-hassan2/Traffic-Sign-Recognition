# Traffic-Sign-Recognition

This repository contains a CNN model for traffic sign classification. It takes an input image of a traffic sign and predicts the correct sign class. The model was trained on the German Traffic Sign Recognition Benchmark (GTSRB) dataset and achieves up to 95% validation accuracy. This project can be useful in driver assistance settings, where traffic sign recognition can improve driver awareness and support safer driving.
